# TASK-1-NWANNE-EMMANUELLA-CHIJINDU
Repository for UIUX Design Project 1 on The Empathy Map
Project Description

This project was completed as part of DeCodeLabs Internship program. The task was to create an empathy map as a UI/UX designer in order to gain a deeper understanding of users and design from a user-centered perspective.

An empathy map is a UX research tool used to organize user insights by focusing on what users say, think, feel, and do. This project demonstrates my ability to analyze users and translate their experiences into meaningful design insights.

Project Objective

The objective of this project was to:

Understand the target user and their needs
Identify user pain points, motivations, and goals
Practice empathy-driven design thinking
Build a strong foundation for future design decisions
My Approach

To complete this project, I:

Identified the target user for the product
Analyzed user behaviors and possible frustrations
Mapped user thoughts, feelings, actions, and spoken feedback
Organized these insights into the four empathy map sections

This approach helped ensure that design decisions are informed by real user perspectives rather than assumptions.

Outcome

The final empathy map provides a clear overview of the user experience, highlighting key challenges and emotional triggers. These insights can be used to guide better UI/UX decisions and improve the overall user experience of the product.
